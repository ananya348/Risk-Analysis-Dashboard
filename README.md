# 🚀 Enterprise Risk Analysis Dashboard

## 📊 Project Overview

This project focuses on analyzing financial transactions to identify **risk patterns** and **fraudulent activities** using data analytics and visualization techniques.

It simulates a real-world **Governance, Risk, and Compliance (GRC)** use case.

## 🛠️ Tech Stack

* 📊 Excel → Data Cleaning & Feature Engineering
* 🗄️ MySQL → Data Storage & SQL Analysis
* 🐍 Python → Data Analysis & Visualization
* 📈 Power BI → Interactive Dashboard
* 🤖 Machine Learning → Fraud Prediction Model

---

## 📁 Dataset Features

* Transaction Type
* Transaction Amount
* Region
* Payment Method
* Fraud Flag
* Likelihood
* Impact
* Risk Score
* Risk Level

---

## 📊 Key Insights

* 🚨 Fraudulent transactions show **higher risk scores**
* 💳 Certain payment methods have **higher risk exposure**
* 📈 High-value transactions contribute to **increased risk**
* 🌍 Risk distribution varies across regions
